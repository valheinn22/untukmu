Cara Hosting Website Ini di GitHub

  1. Salin semua kode halaman ini (Ctrl+A lalu Ctrl+C)
  2. Buat file baru di komputer Anda, beri nama index.html
  3. Buka file tersebut dengan text editor, tempel kode yang disalin (Ctrl+V)
  4. Simpan file
  5. Buka github.com dan buat akun (jika belum punya)
  6. Buat repository baru dengan nama username.github.io (ganti "username" dengan username GitHub Anda)
  7. Upload file index.html ke repository tersebut
  8. Pergi ke Settings ‚Üí Pages ‚Üí pilih branch "main" ‚Üí Save
  9. Tunggu beberapa menit, lalu akses https://username.github.io
